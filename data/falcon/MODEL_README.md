# 🦅 Falcon V0.4.0 — 模型说明书

## 一句话描述
**Falcon每天给标普500的476只股票打分，选出财务最健康、分析师最看好的5只，模拟买入持有30天。**

## 找什么样的股票？
Falcon找的不是"短期涨最快"的股票，而是**基本面最扎实的公司**：
- 💰 **赚钱能力强**（利润率高、现金流好、负债低）— 权重70%
- 📈 **分析师看好**（盈利预期被上调、分歧度低）— 权重20%
- 🏭 **运营效率高**（ROE高、资产回报好）— 权重10%

**不看**：技术指标（RSI/动量等）、新闻情绪、K线形态。已验证这些对大盘股无增量alpha。

## 信号等级
| 信号 | 条件 | 含义 |
|------|------|------|
| 🟢🟢 | 综合分≥0.65 且 排名前5% | 强烈推荐买入 |
| 🟢 | 综合分≥0.65 且 排名前20% | 值得关注 |
| 🟡 | 综合分≥0.50 | 及格，不推荐 |
| 🔴 | 综合分<0.50 | 不推荐 |

## 交易规则
- **买入**：🟢🟢信号 + 不在当前持仓 → 等额分配现金
- **卖出**：持有满30天 或 亏损达15%（止损）
- **仓位**：最多5只，留5%现金缓冲
- **执行**：纽约开盘9:30 ET，通过Alpaca Paper Trading

## 数据来源
| 数据 | 来源 | 更新频率 | 用途 |
|------|------|---------|------|
| 财务报表 | FMP API | 季度 | PE/PB/利润率/负债率 |
| 分析师预期 | FMP API | 每周 | 盈利/收入修正 |
| 股价 | Massive.io | 日频 | 技术指标(备用) |
| 交易 | Alpaca | 实时 | Paper Trading |

## OOS回测表现

- **OOS Sharpe**: 1.836（测试集，不降反升）
- **OOS标准**: 4/4通过✅
- **策略**: 固定持有30天，止损-15%

## 已知限制
1. **数据止于2024-12-31**：features_v02.parquet未更新到2025，当前信号基于2024年底数据
2. **只覆盖SPX 476只**：不含$1-10微盘股
3. **基本面季度更新**：财报之间的变化不会被捕获
4. **无市场择时**：无论牛熊都满仓，熊市回撤可能较大

## 自动化流程
```
每日开盘(HKT 21:30):
  falcon_score.py --universe spx → 生成信号
  falcon_trade_exec.py → 执行交易 + Telegram报告

每日收盘(HKT 04:30):
  falcon_trade_exec.py --report → 持仓快照

每5分钟(盘中):
  falcon_observer.py → 监控异动 → alert → Telegram
```

## 文件说明
| 文件 | 用途 |
|------|------|
| `falcon_v031_scored_YYYYMMDD.json` | 每日评分结果，含排名/分数/信号 |
| `observer_state.json` | 实时监控股票的价格/成交量 |
| `ref_cache.json` | 缓存前日收盘价(非交易时段用) |
| `trades/trade_journal.jsonl` | 交易日志(每笔一行，含买入理由) |
| `trades/positions.json` | 当前持仓记录 |
| `trades/report_YYYYMMDD.json` | 每日交易报告 |
| `macro_snapshot.json` | 大盘宏观指标快照 |
