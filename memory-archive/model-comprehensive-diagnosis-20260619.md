## 一、时间线与模型演进全景

### Phase 1: V3 公式时代 (6/18 之前)
**设计初衷**: 简单可解释的规则系统，110分制评分
- 趋势30分 + 动量25分 + MACD25分 + 偏离10分 + RSI10分 + 52周
- **实战**: 夏普0.3-0.77，DD-13.7%，年化+8-12%
- **优点**: 无过拟合、可解释、DD控制极好
- **缺点**: 收益低、无学习能力

### Phase 2: V4 机器学习突破 (6/18 白天)
**设计初衷**: 用ML替代规则，提升收益
- NeuralNet分类器 → 78.4%准确率是假的 → 实际夏普0.66
- 发现排序能力 > 分类能力 → Top-N策略
- XGB+LGB等权集成 → **夏普1.16**（2.5年测试期）
- **实战**: 年化+43.5%，DD-29.2%

### Phase 3: 深度研究 (6/18 晚)
**设计初衷**: 41个实验找最优模型+参数
- S&P500，514只，2016-2026
- CatBoost平均DD最优（-2.47%），LGB夏普最高（1.72）
- 10天持有期最优，SL-10%止损最优
- 加权集成夏普1.57，DD-20.1%
- **⚠️ 全部是2.5年测试集结果，未做Walk-Forward**

### Phase 4: V3 vs V4 vs V3+V4 (6/18 晚)
**设计初衷**: 找出V3和V4的最佳组合方式
- 34个实验，10年Walk-Forward验证
- **关键反转**: 单次训练V3+V4夏普1.37 → WF后只有0.90
- 纯V4 WF夏普1.13，V3过滤后反而降到0.90
- **结论**: 不做V3过滤，V3只做风控监控

### Phase 5: 特征工程+截面排名 (6/19 凌晨)
**设计初衷**: 用截面排名特征突破1.13的天花板
- 16维截面特征单独用: **夏普1.233**（S&P500）
- 原V4特征+截面特征反而变差（互相干扰）
- 3天持有期DD从-82.4%改善到-64.9%

### Phase 6: 全市场+风控 (6/19 早上)
**设计初衷**: 消除幸存者偏差，加入风控
- 数据: 514只→2413只（含退市/失败公司）
- S&P500夏普1.233 → 全市场夏普0.521（裸）
- 加入波动率目标+回撤缩放 → **蓝盾夏普0.949，DD-22.6%**
- 绿箭夏普0.739，DD-26.7%

## 四、我看到的核心问题（按严重性排序）

### 🔴 问题1：最好的特征方案从未被完整验证

我们在S&P500上发现了16维截面特征的威力（夏普1.233），但**从未**把它放到全市场+风控的框架下验证。最终生产方案用的是base_28+cs8的混合方案（夏普0.949），这不是最优的特征组合。

```
已验证路径:
  16维CS (S&P500, 无风控): 夏普1.233, DD-64.9%  ✅ 已验证
  base_28+cs8 (全市场, 有风控): 夏普0.949, DD-22.6%  ✅ 已验证
  
未验证:
  16维CS (全市场, 有风控): 夏普???, DD???  ❌ 从未测试！
```

**这是最大的遗漏。** 截面特征+全市场+风控，可能是真正的最优组合。

### 🔴 问题2：成本敏感性分析疑似有bug

看`v4_10year_validation.json`，无论cost_per_trade从0到0.5%，**夏普和年化完全不变**。这明显是bug——回测代码没有正确扣除交易成本。

同时黑板上估计"年交易2000笔，年化成本100%"，但这个数字也没有被实际计算验证。

**两个数字互相矛盾：一个说成本0%，一个说成本100%。我们需要一个真实的成本分析。**

### 🔴 问题3：从S&P500到全市场，夏普暴跌但方案也在退化

```
S&P500:    截面16维    → 夏普1.233
全市场:    base_28+cs8 → 夏普0.949 (加了风控)
全市场裸:  base_28+cs8 → 夏普0.521
```

夏普从1.233降到0.949，看起来是幸存者偏差的代价。但同时**特征方案也从16维CS退化到了base_28+cs8**。我们不知道真正的全市场夏普应该是多少。

### 🟡 问题4：CatBoost被放弃得太早

深度研究报告显示CatBoost的选股质量最好：
- 平均DD仅-2.47%（LGB是-4.45%）
- Calmar比率2.51最高
- 在10年WF中Cat Top-15 7d夏普0.761

但因为LGB的短期夏普更高，CatBoost被完全放弃了。问题是：
- Cat的DD控制更好 → 风控叠加后可能反超LGB
- Cat的平均DD低 → 说明选股质量高，退出信号好
- 我们从未在全市场+风控条件下测试过CatBoost

### 🟡 问题5：交易成本可能吃掉大部分收益

```
蓝盾 Top-20, 5天持有:
- 每5天轮换: 卖20只+买20只 = 40笔
- 年交易次数: 50次/年 × 40笔 = 2000笔
- 每笔成本 0.05%（佣金+滑点）
- 年化成本: 2000 × 0.05% = 100%
```

年化收益才28%，成本可能要50-100%。**策略在实盘中可能是亏钱的。**

解决方案很简单——延长持有期到10-20天——但我们从未正式验证。

### 🟡 问题6：风控的滞后性

用过去20天波动率调整仓位。真实崩盘时：
- VIX从15飙到80只需5天
- 我们的模型会在暴跌**之后**才减仓
- -22%DD已经发生了，风控才开始工作

这不是"改进风控参数"能解决的问题——是结构性缺陷。

### 🟡 问题7：参数优化可能过拟合

Top-N、vol_target、DD阈值都是在同一批Walk-Forward折上选出来的。相当于用测试集选参数，结果会偏乐观。

### 🟢 问题8：绿箭命中率暴跌

从V9的6%暴跌到V10的1.1%。年化25.8%可能来自少数几次大赚，不稳定性极高。

## 五、根因分析：为什么"越来越差"

```
阶段1: V4集成 2.5年测试  → 夏普1.16  (测试集乐观)
阶段2: 10年WF验证       → 夏普1.13  (衰减3%)
阶段3: 全市场验证       → 夏普0.521 (幸存者偏差暴露)
阶段4: 加风控          → 夏普0.949  (风控改善DD，但夏普仍<1)
```

**衰减的三个来源：**
1. **测试集过拟合** (1.16→1.13): 小问题，-3%
2. **幸存者偏差** (1.13→0.521): 大问题，-54%
3. **特征方案退化** (1.233→0.949): 可避免的问题，-23%

前两个是不可避免的——我们在发现真相。但第三个是操作失误：最优的特征方案从未在全市场+风控条件下验证。

   - 缺少基本面alpha（盈利质量、估值修复）
   - 缺少另类数据（情绪、资金流、事件驱动）
   - 技术面ML在美股的alpha可能已经很薄了

## 七、结论

**不是做的越来越差，是做的越来越真实。**

从2.5年测试集的虚高夏普，到10年WF的衰减，到全市场的幸存者偏差暴露——每一步都是在接近真相。这个方向是对的。

但过程中有两个可避免的错误：
1. 最优特征方案（16维CS）从未在全市场+风控条件下验证
2. 交易成本从未被正确计算

**先把这两个补上，再决定下一步。**
