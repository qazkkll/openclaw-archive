# Plan A: 美股择时策略设计方案
> 2026-06-19 | 设计阶段

---

## 一、为什么从选股转向择时

### 现状
```
蓝盾选股(全市场): 扣成本后夏普 ≈ 0.3-0.5
S&P500买入持有:  同期夏普 ≈ 0.5-0.7
```

### 根因
- 技术面ML选股在美股的alpha极薄（机构已充分挖掘）
- Top-20×5天持有=年2000笔交易，成本吃掉大部分收益
- 选股策略的alpha来源（价格模式）和择时信号（趋势/波动）本质上是不同维度

### 择时的优势
| 维度 | 选股 | 择时 |
|------|------|------|
| 交易频率 | 每5天全换(50次/年) | 月频或更低(6-12次/年) |
| 交易成本 | 年化50-100% | 年化<1% |
| 信号来源 | 个股价格噪声 | 宏观趋势/波动率(信噪比高) |
| 幸存者偏差 | 严重 | 无(SPY本身) |
| 执行难度 | 每天选20只+下单 | 月度调仓一次 |

## 二、策略框架

### 核心思路
**Meb Faber 10月均线 + VIX波动率叠加 + 多资产轮动**

三个层次信号组合：
1. **趋势层**: SPY价格 vs 200日均线（核心信号）
2. **波动层**: VIX水平和变化率（辅助信号，调整仓位大小）
3. **资产层**: SPY/TLT/GLD三资产轮动（分散化）

### 信号定义

#### 信号1: 趋势过滤（主信号）
```
SPY月线收盘 > 10月SMA(≈200日SMA)  →  做多
SPY月线收盘 < 10月SMA              →  持有现金/债券
```

Meb Faber原始论文验证：10月SMA在1901-2012年间
- 买入持有年化: 9.5%, DD -56%
- 趋势过滤年化: 10.5%, DD -22%
- **夏普提升 +30%, DD降低60%**

#### 信号2: VIX波动层（辅助信号）
```
VIX 20日均线 < 18        →  正常仓位（100%）
VIX 20日均线 18-25       →  减仓（50-70%）
VIX 20日均线 > 25        →  最小仓位（20-30%）
VIX 20日均线 > 35        →  清仓（0%）
```

Alpha Architect研究：VIX叠加趋势跟踪在双资产模式下夏普+0.09

#### 信号3: 多资产轮动（分散化）
```
每月末评估三个资产的3个月SMA > 10个月SMA:
  - SPY（美股）
  - TLT（长期国债）
  - GLD（黄金）
等权分配给所有"上涨"的资产，现金为0时持现金
```

Meb Faber 2015论文验证（1971-2015）：
- 动量轮动: 年化13.12%, 夏普0.63, DD -21.4%
- 买入持有等权: 年化9.88%, 夏普0.50, DD -21.0%
- **纯美股: 年化10.46%, DD -51.0%**

## 三、具体实施方案

### 方案A: 简单版（推荐起步）
```
资产: SPY（美股）+ TLT（债券）+ GLD（黄金）
信号: 3月SMA > 10月SMA → 持有该资产
仓位: 等权分配给所有信号为正的资产
调仓: 月末最后一天
交易成本: 每年约6-12次交易
预期: 年化8-12%, 夏普0.6-0.8, DD < -25%
```

### 方案B: 增强版（叠加VIX）
```
在方案A基础上:
1. VIX < 18: 正常等权
2. VIX 18-25: 减仓到70%，多出30%持现金
3. VIX > 25: 减仓到30%，70%现金
4. VIX > 35: 全部现金
预期: 年化7-10%, 夏普0.7-0.9, DD < -18%
```

### 方案C: 纯美股版（最简单）
```
资产: 仅SPY
信号: SPY月线 > 200日SMA → 满仓SPY
      SPY月线 < 200日SMA → 100%现金
调仓: 月末
交易次数: 年均4-8次
预期: 年化8-11%, 夏普0.6-0.8, DD < -25%
```

## 四、回测验证计划

### 数据
- SPY/TLT/GLD: 2000-2026 (26年，覆盖3次大崩盘)
- VIX: 2004-2026 (22年)
- 免费数据源: yfinance（已验证可用）

### 验证方法
1. **全样本回测**: 2000-2026，看长期表现
2. **Walk-Forward**: 5年训练+1年测试，滚动前进
3. **交易成本敏感性**: 0/0.05%/0.1%每笔
4. **参数稳定性**: 测试8月/10月/12月均线，验证不依赖特定参数

### 评估指标
| 指标 | 目标 | 说明 |
|------|------|------|
| 夏普 | > 0.7 | 扣成本后 |
| 最大DD | < -25% | 可承受 |
| 年化收益 | > 8% | 扣成本后 |
| 交易次数 | < 12次/年 | 控制成本 |
| 参数稳定性 | 各参数差异<20% | 不过拟合 |

## 五、与现有系统的关系

### 并行运行，不替代
```
蓝盾V4(选股):  继续在实盘运行，观察真实表现
Plan A(择时):  新建独立回测，验证后作为备选方案

如果Plan A夏普>蓝盾扣成本后夏普 → 切换到Plan A
如果Plan A夏普<蓝盾 → 保留蓝盾，Plan A作为风控参考
```

### 风控融合
即使不切换到Plan A，200日均线信号也可以作为蓝盾的风控层：
```
SPY > 200日SMA → 蓝盾正常运行
SPY < 200日SMA → 蓝盾仓位减半或清仓
```
这比当前的"过去20天波动率"更前瞻性。

## 七、预期结论（预判）

根据学术研究和历史数据：
- 纯美股200日均线择时: 夏普约0.6-0.8（vs 买入持有0.5-0.7）
- 多资产轮动: 夏普约0.63-0.77
- VIX叠加: 额外+0.03-0.09
- **预期真实夏普: 0.6-0.9**

这可能不如蓝盾回测的0.949，但**扣成本后的真实表现可能反超**。
因为择时策略年化交易成本<1%，而选股策略50-100%。
