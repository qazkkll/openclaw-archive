🦅 Falcon 是什么？

Falcon 是一个自动选股系统。它每天收盘后自动给标普500的476只股票打分，选出最值得买的5只，告诉你什么时候买、什么时候卖。

核心思路：用公司财务数据判断"值不值"，用分析师预期判断"市场怎么看"，两者结合选出既有实力又被看好的股票。

📊 打分逻辑（三个维度）

维度权重通俗解释举例
财务健康
Fundamentals		70%公司赚不赚钱、贵不贵、财务有没有风险。看PE、利润率、负债率等20个指标。苹果PE=28，利润率26%，负债低 → 高分
分析师态度
Analyst		20%华尔街分析师最近上调还是下调了盈利预期。上调=看好，下调=看空。NVDA分析师连续3个月上调预期 → 高分
关键指标
Key Metrics		10%ROE（赚钱效率）、ROA（资产利用效率）等核心经营数字。ROE=30%说明每投入1块钱赚3毛 → 高分

为什么财务占70%？因为历史回测证明，公司基本面是长期股价最可靠的驱动力。分析师修正提供短期催化剂，两者互补。

🔄 工作流程（每天自动）

16:00 纽约 → 美股收盘 → falcon_score.py → 拉取最新财务数据 → 476只股票打分排名 → 输出Top 5信号文件
次日 9:30 → Observer → 监控这5只股票实时价格 → VWAP择时（低于均价时买入） → Alpaca纸交易下单
持有30天 → 到期卖出 或 亏15%止损 → 记录盈亏 → 周六自动复盘

📈 信号等级怎么看？

信号含义操作
🟢🟢排名前5%，强烈推荐可以立即买入
🟢排名5-20%，推荐可以关注，等回调再买
🟡排名20-50%，一般不建议买入
🔴排名后50%，不推荐不要买

⚠️ 已知限制

  • 财务数据更新频率：季度更新（财报发布后），非实时。财报之间的数据可能过时。
  • 样本外表现：2024下半年 Sharpe=1.84，胜率62%。但过去不代表未来。
  • 交易成本：每次买卖约0.1%成本，频繁交易会侵蚀利润。
  • 技术面因子已关闭：历史验证无增量alpha，只用财务+分析师。
  • 情绪因子待部署：FinBERT新闻情绪数据还在补全中。

🧪 模型验证方法

Falcon用Walk-Forward验证——把历史数据切成三段：

阶段时间段干什么类比
训练集2023.1 - 2024.3模型学习规律高考前刷真题
验证集2024.4 - 2024.6调参数模拟考试
样本外2024.7 - 2024.12最终考试高考真题（模型从没见过）

只有样本外的成绩才算数。训练集表现好不代表模型靠谱——可能是"背答案"（过拟合）。
