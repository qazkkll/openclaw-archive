## 关键发现

### 1. V3公式的价值：安全网，不是排序器

V3的110分制（趋势30+动量25+MACD25+偏离10+RSI10+52周）是一个**二值化评分**——每个维度要么得满分，要么0分。这种设计：
- ✅ 擅长过滤明显差的股票（V3<60的股票确实表现差）
- ❌ 不擅长在好股票中做精细排序（所有V3高分股票得分相同）

### 2. V4 LGB的价值：精细排序器

LGB用35维特征学习连续的预测值，擅长在"已经不错的股票"中找出最好的。但它对**明显差的股票**没有足够的"拒绝能力"。

### 3. 组合的逻辑：V3 gatekeeper + V4 ranker

```
全市场514只 → V3≥60过滤（约300只）→ LGB排序 → Top-15买入
```

- V3负责"删坏的"（规则明确，不过拟合）
- LGB负责"选好的"（连续信号，有区分度）
- 8%止损负责"控制尾部风险"

### 4. 加权融合失败的原因

所有权重（10%-90%）的加权融合都显示相同的低表现（夏普0.51）。原因是**尺度不兼容**：
- V3评分：0-110离散值，均值65
- LGB预测：连续小数值（5日收益率预测）

简单的加权平均让大尺度的V3主导了排序，相当于退化成了V3纯规则。**正确做法是先归一化到同一尺度**，但实验已经证明"过滤+排序"比"加权融合"更有效。

### 5. V3作为特征无效

把V3评分加入LGB的35维特征中，夏普从1.06→1.06（无变化）。说明V3公式捕捉的信息**已经被LGB的35维特征完全覆盖**——V3的趋势分≈ret_5d+ma_20_ratio，动量分≈ret_5d+ret_20d，等等。


### ⚠️ 重要警告

以上是**单次训练回测**结果。基于之前的经验：
- 纯LGB单次训练→walk-forward衰减约35-40%
- V3公式不衰减（无训练过程）
- V3+V4组合的衰减率**未知**

**需要跑walk-forward验证**才能确认真实夏普。估计walk-forward夏普可能在0.9-1.1范围。

## 失败的方案（排除）

1. **V3+V4加权融合**：尺度不兼容，退化成V3纯规则
2. **V3作为LGB额外特征**：信息冗余，无提升
3. **V3过滤训练集**：训练样本减少35%，泛化能力下降
4. **V3+V4择时**：择时信号与V3过滤重叠，收益下降
