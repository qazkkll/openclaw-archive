# 蓝盾V4 深度研究报告 — 2026-06-18 20:30

## 实验概述
- **数据**: S&P 500成分股，514只，2016-2026，1,213,017有效样本
- **模型**: XGBoost / LightGBM / CatBoost / 加权集成(0.4/0.3/0.3)
- **训练**: 2016-2021 | 验证: 2022-2023 | 测试: 2024-01~2026-06 (2.5年)
- **特征**: 37维（收益率/波动/RSI/MACD/布林/量能/均线/52周/截面排名/市场宽度）
- **实验数**: 41个（4模型×3Top-N×3持有期 + 止损/择时组合）

---

## 🏆 核心结论

### 一、模型层：CatBoost是平均回撤之王

| 指标 | Cat Top-20 10d | LGB Top-10 10d | 加权集成 SL-10% |
|------|---------------|----------------|----------------|
| **夏普** | 1.57 | 1.72 | 1.57 |
| **年化** | +37.9% | +71.0% | +53.2% |
| **最大DD** | **-15.1%** | -33.1% | -20.1% |
| **平均DD** | **-2.47%** | -4.45% | -3.53% |
| **胜率** | 57.7% | 56.2% | 55.7% |
| **Calmar** | **2.51** | 2.15 | 2.65 |

**结论：CatBoost选股质量最高**。平均DD仅-2.47%，说明模型选出的股票平均回撤很小，退出预期好。这就是Andy说的"模型层"问题的解。

### 二、决策层：止损+择时控制最大DD

| 配置 | 夏普 | 年化 | 最大DD | 平均DD |
|------|------|------|--------|--------|
| 基线（无止损） | 1.29 | +44.8% | -32.4% | -4.09% |
| SL-5% | 1.32 | +37.7% | **-16.8%** | -3.41% |
| SL-8% | 1.45 | +46.6% | -20.6% | -3.51% |
| SL-10% | 1.57 | +53.2% | -20.1% | -3.53% |
| 择时+SL-8% | 1.45 | +36.5% | -20.1% | **-3.05%** |

**结论：**
- SL-8%是最佳平衡点：最大DD从-32.4%→-20.6%，年化反而提升（截断亏损+保留盈利）
- SL-5%过度止损：378次止损（1293笔中29%），胜率降至51.7%，交易成本太高
- 市场择时效果有限：仅在SL-8%基础上提供微小改善

### 三、关键发现

#### 1. 10天持有期最优
- 10d策略的夏普普遍高于5d和7d
- 原因：更长持有期让趋势充分发展，减少频繁换仓的成本
- Cat Top-15 10d: 夏普**1.74**，是所有41个实验中最高

#### 2. 平均DD的含义（Andy的核心关注）
- 平均DD = 每次回撤事件的平均深度
- CatBoost: -2.47%~ -3.09% → 模型选出的股票"跌也跌不深"
- XGB: -3.23%~ -4.95% → 选股质量较差
- LGB: -3.18%~ -5.07% → 最高夏普但选股质量一般

**这验证了Andy的判断：平均DD高 = 模型退出信号差 = 选股质量差**

#### 3. 集成 vs 单模型
- CatBoost单独就比集成效果好（在DD控制上）
- 加权集成被XGB和LGB的高波动"污染"
- **建议：生产环境用CatBoost单模型，或CatBoost为主+LGB辅助**

#### 4. 市场择时几乎无效
- 基线: 夏普1.29 → 择时后: 夏普1.22（反而降低）
- 原因：截面宽度（breadth）作为市场信号太粗糙，不如直接让模型学习

## ⚠️ 注意事项

1. **无SPY/VIX数据**：数据文件仅含514只个股，缺少市场指数。之前的特征重要性分析（spy_ret5占7%）不可复现
2. **测试期仅2.5年**：2024-2026是牛市+高波动期，结果可能有样本选择偏差
3. **没有交易成本**：实际交易有佣金+滑点，会降低真实夏普
4. **CatBoost的10d持有期**：实际是每10个交易日轮换一次，不是持有10天就卖
