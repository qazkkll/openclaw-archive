## 一、现有模型分析

### 蓝盾V3（公式模型）

| 维度 | 分值 | 逻辑 |
|------|------|------|
| 趋势 | 30分 | MA5/MA20/MA60排列 |
| RSI | 20分 | 14日RSI |
| 均线 | 20分 | 价格vs均线位置 |
| 52周 | 15分 | 52周高低位 |
| 量能 | 10分 | 5日/20日量比 |
| 波动 | 5分 | 日涨跌幅 |

**优点：** 无依赖、可解释、快速
**缺点：** 阈值固定、无学习能力

### 绿箭V9（XGBoost模型）

**特征：** 50维（均线、动量、波动、成交量）
**目标：** 5天涨50%概率

**优点：** 机器学习、可量化信心
**缺点：** 特征依赖人工、时序处理有限

## 二、改进模型（V10）

### 新增特征

| 特征 | 重要性 | 说明 |
|------|--------|------|
| 波动率 | 0.208 | 最重要！高波动=大机会 |
| MA60比率 | 0.174 | 长期趋势 |
| MA20比率 | 0.140 | 中期趋势 |
| MA交叉 | 0.131 | 趋势确认 |
| RSI | 0.122 | 超买超卖 |
| MA5比率 | 0.122 | 短期趋势 |
| 动量 | 0.103 | 涨跌速度 |

### 训练结果

- **样本数：** 4,796
- **正样本比例：** 20.31%
- **准确率：** 79.79%

### 关键发现

1. **波动率是最重要特征** — 高波动股票更容易出现大行情
2. **多时间框架有效** — MA5/MA20/MA60组合比单一均线好
3. **RSI仍然有效** — 但需要结合其他指标
