```
方法: XGBoost/LightGBM/CatBoost, 5天持有, Top-15
结果: WF夏普1.13, 年化+48%, DD-57%
关键实验:
  - 单次训练夏普2.22 → WF后1.13（衰减49%，过拟合严重）

教训:
  1. 单次训练回测会欺骗你
  2. V3做风控比做选股好
  3. 止损在ML策略中拖累alpha
  4. 概率校准是根因——模型输出挤在70%无区分度
归档: models/us/archive/v3_v4_legacy/
```

## 二、V5设计哲学

### 为什么是120天？
```
5天:  噪音大, 交易成本高(900笔/年), alpha薄
20天: 信号改善, 成本仍高(225笔/年)
60天: 信号清晰, 成本可控(75笔/年)
120天: 信号最干净, 成本最低(36笔/年), 夏普最高
```
学术支撑: 1-12月动量因子是美股最稳健的异象之一，年化溢价8-12%。

### 为什么不用截面排名特征？
```
S&P500上: 截面16维夏普1.233 > 全特征1.21
但: 截面特征在全市场+风控下从未验证
结论: 在S&P500上，全特征已经够好，截面特征提升不大
```

### 为什么不用CatBoost？
```
R1对比: XGB夏普1.21 vs CatBoost夏普1.19
差距很小(0.02)，XGB CUDA更快
结论: 用XGB，简单可靠
```

### 为什么不用非对称损失？
```
R2对比: 标准MSE夏普1.21 vs 非对称(3x)夏普1.20
几乎没有差别
结论: 标准MSE够用
```

### 为什么不用集成？
```
R2对比: XGB 1.21 vs XGB+Cat 1.20 vs CatBoost单独 1.19
集成反而略差（CatBoost拉低了整体）
结论: XGB单模型最优
```

## 三、V5模型规格

### 配置
| 参数 | 值 |
|------|-----|
| 算法 | XGBoost |
| 设备 | CUDA (3080 Ti) |
| max_depth | 6 |
| learning_rate | 0.03 |
| subsample | 0.8 |

| n_estimators | 500 (early stopping 50) |
| 持有期 | 120天 |

| 股票池 | S&P500 (514只) |
| 重训频率 | 每6个月 |
| 训练窗口 | 3年 |

### 28个特征

**趋势 (6个)**
- `ma5` — 5日均线
- `ma20` — 20日均线
- `ma60` — 60日均线
- `ma_bias20` — MA偏离度
- `ma_align` — 均线排列(0-2)
- `price_position` — 60日价格位置(0-1)

**动量 (8个)**
- `ret1/5/20/60` — 多周期收益率
- `momentum_6m` — 6个月动量
- `momentum_1m` — 1个月动量
- `mom_divergence` — 动量背离
- `trend_accel` — 趋势加速

**波动率 (4个)**
- `vol20` — 20日波动率
- `vol5` — 5日波动率
- `vol_ratio` — 量比
- `vol_change` — 波动率变化

**技术指标 (7个)**
- `rsi14` — RSI
- `rsi_change` — RSI变化
- `macd/macd_signal/macd_hist` — MACD三件套
- `bb_std/bb_width/bb_pos` — 布林带三件套

**质量 (1个)**
- `ret_quality` — 收益/风险比

### 特征重要性排名
```
 1. vol20         波动率      ← 最重要！
 2. ma60          60日均线
 3. bb_std        布林标准差
 4. momentum_6m   6个月动量
 5. ret60         60日收益
 6. ma5           5日均线
 7. ma20          20日均线
 8. bb_width      布林带宽
 9. macd_signal   MACD信号
10. vol_change    波动率变化
```
**洞察**: 波动率是最重要的特征，不是动量。模型在找"低波动+趋势好"的股票。

## 五、关键教训（心路历程）

### 教训1: 单次训练回测会欺骗你
```
V4单次训练: 夏普2.22
V4 Walk-Forward: 夏普1.13
衰减: 49%
教训: 任何"看起来很好"的结果都需要WF验证
```

### 教训2: 幸存者偏差是隐形杀手
```
S&P500选股: 夏普1.23
全市场选股: 夏普0.52
差异: 54% ← 幸存者偏差
教训: 只包含成功公司的数据会严重高估策略表现
```

### 教训3: 交易频率是成本杀手
```
5天持有: 900笔/年, 成本~27%
120天持有: 100笔/年, 成本~3%
教训: 减少交易频率是提升净收益最直接的方法
```

### 教训4: 模型选择不如持有期重要
```
XGB vs CatBoost: 差距0.02夏普
5天 vs 120天: 差距0.57夏普
教训: 参数优化的边际收益远小于架构决策
```

### 教训5: 特征筛选的收益有限
```
14特征: 夏普1.21
12特征: 夏普1.19
5特征: 夏普1.18
教训: 当特征集已经比较干净时，筛选的提升很小
```

### 教训6: 牛市里什么策略都好使
```
2020年: +124% (COVID反弹)
2023年: +86% (AI牛市)
教训: 牛市里的高收益不代表策略能力强
      真正的考验是2022年熊市(+4.5% vs SPY -17.5%)
```

## 六、已知局限

1. **S&P500有幸存者偏差** — 只包含成功公司，实际效果可能打7折
2. **DD估计不精确** — 重叠窗口导致DD计算有偏差，需独立验证
3. **2020年+124%有运气成分** — COVID反弹是特殊事件
4. **未测试交易成本敏感性** — 当前假设0.03%/笔，实际可能更高
5. **无基本面特征** — 只用技术面，未加入盈利修正/ROE等
