# V9-Lottery README — 绿箭V9 心路历程

## 简版

V9-Lottery = V8的36特征 + V9新增14特征（含开盘跳空/成交量/日内强度）+ 二分类目标（+50%爆不爆）+ 全量2436只x8年OHLCV数据 + 正则化压满。

比V8所有维度都赢。正式定版。

## 详细心路

### V8（2026-06-12）的问题

V8是个回归模型，预测"5天涨多少"。它的训练目标是平滑回归损失，不是识别彩票。这导致：
- V8给所有$1-10票打3-15分（温和分布）
- 前5%只有3.9%是彩票
- 高分段(≥20)只有373票，样本太少不靠谱
- OOS测试两个期抓0个彩票

### 为什么从回归改分类

彩票的本质是稀有事件（池中0.5-1%）。回归模型把资源花在预测"涨3%还是涨5%"的细微差别上，而不是识别"会不会涨50%"。改成二分类后，模型直接学习"什么信号预示着极端涨幅"，资源分配对了。

### V9的四轮迭代

**v1**（295彩票事件，36特征，平衡采样）→ 过拟合，0.99+给所有票

**v2**（722事件，$0.5-20池扩容，50特征）→ 有过拟合，但前向测试16%抽中率vs V8的3-7%

**v3**（235事件，587只带开盘价，heavy正则化）→ ≥0.95精确率25%但样本太少(4票)，统计假象

**v4（定版）**（1300事件，2436只全量x8年OHLCV，50特征，正则压满）→ 76,477样本验证，所有指标确认

### 关键设计决策

1. **不限制候选池价格** — 让模型从数据里自己学"便宜才是彩票土壤"
2. **开盘跳空特征** — 彩票爆发前常有跳空，之前V8没有
3. **成交量特征** — 放量是彩票的必要条件
4. **正则化拉满** — max_depth=4, subsample=0.5, gamma=1, reg_lambda=5，宁可欠拟合不做过拟合
5. **50特征架构** — 36维V8基础 + 14维新特征（量价/势能/反转信号）

## 性能总结

**252天完整回测：** `data/v9_252d_results.json`

| 指标 | V8 | V9 | 提升 |
|:---|---:|---:|:---:|
| 6期Top5彩票命中 | 4% | **6%** | 1.7x |
| 每日命中天数 | 28% | **51%** | 1.8x |
| OOS期(2期) | 3.5% | **11%** | 3.1x |
| ≥90%命中率 | 15%(仅373票) | **14%(1,210票)** | 量3.2x |
| 前5%平均涨幅 | +15.7% | **+23.8%** | 1.5x |
| Top10% >50%率 | 3.6% | **5.8%** | 1.6x |

## 风险边界

- V9 ≥90%的票平均+29.5%，中值+20.8% — 14%会爆+50%
- V9 ≥85%的票平均+18.6%，中值+12.1% — 5%会爆+50%
- V9 <85%的票平均+6~12%，彩票率骤降到2%以下
- **别在<85%的票上做彩票期望**，它们是温和动量不是真彩票
- ⭐5（高确信信号）的票13.95%平均收益，4次彩票命中全在⭐5档
