## 四、关键教训

### 教训1: 排名比回归有效10倍
```
V6回归(预测绝对收益): 相关性0.026 → 废了
V6排名(预测相对排序): 夏普1.44 → 有效
预测"谁比谁强"比"涨多少"容易得多
```

### 教训2: 宏观特征是关键
```
27维纯技术: 夏普0.434
44维(+宏观+基本面): 夏普0.618 (+42%)
→ "市场在涨还是跌"比"个股技术面"更重要
```

### 教训3: 全市场验证才能信
```
S&P500: 夏普1.36(虚高)
全市场: 夏普1.44(真实)
→ 幸存者偏差让S&P500结果不可信
```

### 教训4: 120天持有期是错的
```
V5 120天: 10笔交易 → 统计无意义
V6 20天: 30轮交易 → 统计可信
→ 持有期要平衡"信号质量"和"样本量"
```
