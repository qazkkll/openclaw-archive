## 评分函数详情

```python
score = 0
score += (-ret20).clip(-0.3, 0.3) * 3      # 反转：跌幅越大越好
score += rank(total_net_5d) * 2             # 资金流入：流入越多越好
score += (1 - rank(vol20)) * 2              # 低波动：波动越小越好
score += (rsi_14 < 35) * 1.5               # 超卖：RSI<35加分
score += rank(lg_net_5d) * 1               # 大单：大单流入加分
score += (-ma20_bias).clip(-0.2, 0.2) * 1  # 均线偏离：偏离越大越好
```

## 核心逻辑

1. **反转效应**: A股短期动量是反转信号(负IC)。选跌多的股票。
2. **资金流**: 中单/大单净流入是唯一正IC的技术特征。
3. **低波动**: 低波动股票下行风险小，止损触发率低。
4. **止损截断**: -3%止损截断亏损尾部，创造极端不对称分布。

## 已知限制

1. **止损执行假设**: 回测假设止损精确在-3%执行，实际可能滑点到-4%~-5%
2. **流动性假设**: 未考虑小盘股的流动性限制
3. **生存偏差**: 数据可能不含已退市股票
4. **2023年最弱**: Sharpe仅0.48，策略在低波震荡市表现弱
5. **规则型无自适应**: 评分权重固定，不能随市场环境调整
