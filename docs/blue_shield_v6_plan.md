# 蓝盾V6 技术方案
> 2026-06-19 | 决策：Andy | 执行：Hermes Workers

---

## 一、设计原则

### 从V5学到的教训
1. **标签=持有期是自证循环** → V6标签和持有期分离
2. **10笔交易无统计意义** → 持有期缩短到10-30天，增加样本量
3. **相对排名不够用** → 输出绝对预期收益%
4. **无风控不可实盘** → 每天重新预测+宏观止损层

### 核心理念
```
蓝盾 = 从>$10股票中，找到20天内能涨10-20%的机会
不是选"最好的"，是选"能赚大钱的"
```

## 二、模型架构

### 输入
- 27个V5技术特征（ma/vol/ret/rsi/macd/bb等）
- 截面排名特征（可选，待验证是否提升）
- 基本面特征（可选，从us_ml_feats_v75获取）

### 输出（3个维度）
```
1. 预期收益率：N天后涨X%（绝对值，回归）
2. 最优持有天数：10/15/20/25/30中选日均收益最高的
3. 信号信心度：历史同类型信号的成功率（%）
```

### 训练方式
```
对每个交易日，同时学5个标签：
  y1 = 持有10天收益率
  y2 = 持有15天收益率
  y3 = 持有20天收益率
  y4 = 持有25天收益率
  y5 = 持有30天收益率

最优持有天数 = argmax(预期收益/持有天数)
```

## 三、信心度与信号分级

### 信心度计算
```
对每类信号（如"预期>10%, 持有15天"）：
  历史回溯：过去5年类似信号出现N次
  成功次数：实际收益>预期收益50%的次数
  信心度 = 成功次数 / N × 100%
```

### 信号分级（待测试最优阈值）
```
🟢🟢（精品买入）：信心度 > 阈值A → 马上下单
🟢（趋势买入）：信心度 > 阈值B → 观察确认后买
🟡（观察中）：信心度 < 阈值B → 不买
🔴（清仓）：预期收益转负 或 宏观恶化
```

## 四、决策框架

### 入场
```
🟢🟢 → 马上市价/限价买入
🟢 → 等突破确认（突破20日高点）再买
```

### 持仓管理（每天重新预测）
```
模型每天更新：
  预期收益仍>10% → 🟢持有
  预期收益降到5-10% → 🟡收紧止损
  预期收益<5% → 🟠减仓1/3
  预期收益转负 → 🔴清仓
```

### 分批止盈
```
不是固定百分比止盈，是模型每天告诉你"还有没有空间"
  模型预期仍强 → 继续持有
  模型预期减弱 → 分批卖出
  模型预期转负 → 清仓
```

## 五、宏观/基本面止损层

### 每天附加分析
```
├── VIX状态：正常(<20) / 警戒(20-30) / 恐慌(>30)
├── 利率趋势：降息 / 持平 / 加息
├── 持仓股基本面：财报/评级/事件
└── 综合判断：🟢正常 / 🟡注意 / 🔴撤退
```

### 双红灯触发条件
```
🔴🔴（立即清仓）：
  - VIX > 35 且 持仓股Beta > 1.5
  - 持仓股财报大幅不及预期（营收/利润>-20%）
  - 持仓股被降级到"卖出"
  - 宏观事件（如2020疫情级别）
```

## 七、执行计划

### Worker编队
| # | 角色 | 任务 | 依赖 | 预计时间 |
|---|------|------|------|---------|
| W1 | 实验员 | 构建+训练多标签模型，Walk-Forward验证 | 无 | 15-20min |
| W2 | 分析员 | 基于W1结果，计算不同阈值的历史成功率，找最优🟢🟢阈值 | W1 | 5-10min |
| W3 | 审查员 | 审查W1+W2的结果：检查过拟合/统计有效性/成本敏感性 | W1+W2 | 5-10min |
| 决策Agent | 我 | 整合W1+W2+W3，给Andy最终结论 | 全部 | — |

### 审查标准（W3职责）
```
1. 样本量：每个信号类型至少30笔交易才可信
2. 过拟合检查：训练集vs测试集性能衰减<30%
3. 成本敏感性：扣成本后夏普仍>0.5
4. 年度稳定性：至少4/5年盈利
5. 极端情况：最大DD是多少，能否承受
6. 与V5对比：比V5好在哪，差在哪
```
